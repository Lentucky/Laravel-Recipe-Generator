# Download Recipe Model

The ML model files are stored on Google Drive (too large for GitHub).

## Download Instructions

1. Click this link: [Download Recipe Model](https://drive.google.com/drive/u/2/folders/1C5TzoRdG5kZv8gTnO8g37RoEkTM9GHJo)

2. Download all the files

3. Extract/place them in your project root so your structure looks like:
   recipe-ai/
   ├── fastapi/ ← Paste here
   ├── app/
   ├── resources/  
   ├── recipe_embeddings.npy ← Paste here
   ├── recipes_clean.csv ← Paste here
   └── (other files)

4. Run the project by running: .\start.bat
